# Philosophy (reference doc)

This skill exists only to bundle `PHILOSOPHY.md` alongside the harness workflow skills so
their "PHILOSOPHY §N" pointers resolve. After install, the skills land as siblings under
`.opencode/skills/`, so a skill at `.opencode/skills/<name>/SKILL.md` reaches this file at
`../philosophy/PHILOSOPHY.md`.

When a workflow skill references a section like `PHILOSOPHY §11`, open
[`PHILOSOPHY.md`](./PHILOSOPHY.md) and read the matching `## §11.` section. The section index
is near the top of that file.
